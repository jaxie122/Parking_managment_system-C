Parking Managment System
This is a basic program designed to calculate and record parking fees for different city zones. It functions as a digital logbook for tracking vehicle stay durations and total costs.

Features:
    Automatic Calculation: The program calculates the total cost based on the specific hourly rate of the chosen zone.
    Data Storage: Every transaction is automatically saved to a text file for future reference.
    Input Checking: It ensures that car numbers use only uppercase letters and digits, and that stay durations are positive numbers.

Pricing per Hour:
    Red Zone: 2.00 RON
    Yellow Zone: 1.50 RON
    White Zone: 1.00 RON

How to Use the Program:
    Start the application: You will be presented with a numbered menu.
    Select Option 1 to Pay: 
        Enter the car plate number using only uppercase letters and numbers.
        Type the zone name (red, yellow, or white).
        Enter the number of hours parked.
    Select Option 2 for History: This displays a list of all recorded transactions from the file.
    Select Option 3 to Exit: This closes the program.

File Information:
All data is stored in a file named parking_data.txt. This file is located in the same folder as the program and can be opened with any standard text editor to view the history manually.
